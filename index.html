WHAT IS FRETNOODLE?

fretnoodle aims to create innovative apps and games to make learning guitar easier,engaging and fun. Check out our tools below!

Fretboard Visualizer

The fretnoodle visualizer simply maps out any inputed note all across the guitar fretboard. This simple yet powerful study tool allows players to visualize any inputed triad, arpeggio or scale all over the neck thus opening up a world of sonic possibilities for improvisation and composition.

Lets GO

MEMORIZE THE FRETBOARD

A simple game to help you memorize all the intervallic positions and notes across the fretboard. Playing this game will definitely make sure all the notes and intervals of the fretboard are burned into your memory

Let's Go

TIM'S TUNER

A simple yet accurate tuner using machine learning pitch detection. This tuner is built for an alternative tuning system designed by Tim, A beautiful tuning system that puts you in state of flow for beginners and professionals
