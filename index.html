WHAT IS FRETNOODLE VISUALIZER?

The fretnoodle visualizer simply maps out any inputed note all across the guitar fretboard.This simple yet powerful study tool allows players to visualize any inputed triad,arpeggio or scales all over the neck thus opening up a world of sonic possibilities for improvisation and composition.

Input any chord/arpeggio/scale and map it across the board to explore all the shapes and ways of playing them..

Input different chords or scales in a progression and observe the transition animations to help visualize how one shape morphs into another

Use the Paint feature (double left click) to highlight different regions and make notes. Extremely useful for teachers explaining concepts to students online.

-->

INSTRUCTIONS

  1. Click on the fretboard layout to input a triad/arpeggio/scale/chord (the computer will treat the first note you input as the root note)
  2. Press 'input new chord' button to input the next triad/arpeggio/scale/chord
  3. Press the 'start animation' button to visualize the whole fretboard
  4. Press 'Next chord/Previous chord' to visualize how the shapes transition and morph into new shapes
  5. Press 'show intervals' button to see the colour coded scale degrees across the board
  6. Press 'start again' to input a new chord progression
